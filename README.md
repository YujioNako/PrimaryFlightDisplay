# PrimaryFlightDisplay
A Primary Flight Display simulator based on ESP32

<big><b>声明：本程序及硬件组装方式中含有AI生成内容，本人不保证这些内容能在任何地方正确工作，请您知悉！</b></big> 

## 需要模块
|模块型号|模块用途|
| ------- | ------- |
|ESP32|计算核心|
|BNO055|姿态传感器|
|BMP280|气压传感器，提供基于气压的高度|
|BN-220|GPS模块，提供基于GPS的速度和高度|
|DFPlayer Mini|MP3模块，提供播放警告音的能力|
|EC11 旋转编码器|旋钮按钮一体控制|
|TFT屏幕（ST7735驱动）|屏幕|

## 接线方法
### I2C通信组件
|模块|模块引脚|ESP32 引脚|备注说明|
| ------- | ------- | ------- | ------- |
|BNO055 (九轴姿态)|SDA|GPIO 21|数据线|
||SCL|GPIO 22|时钟线|
||VCC / VIN|3.3V|3.3V 供电|
||GND|GND|共地|
|BMP280 (气压高度)|SDA|GPIO 21|数据线|
||SCL|GPIO 22|时钟线|
||VCC|3.3V|供电|
||GND|GND|共地|

<b>注意：BMP280模块若为6pin，建议CSE/CSB接3.3V，SDO/SDC接地，避免模块失效</b>

### UART串口通信组件
|模块|模块引脚|ESP32 引脚|备注说明|
| ------- | ------- | ------- | ------- |
|BN-220 GPS|TX (发送)|GPIO 16 (RX2)|模块的 TX 接单片机的 RX。GPS 数据入口。|
||RX (接收)|GPIO 17 (TX2)|模块的 RX 接单片机的 TX。向 GPS 发送 5Hz 配置指令。|
||VCC|3.3V 或 5V|根据模块具体型号支持|
||GND|GND|共地|
|DFPlayer Mini|RX (接收)|GPIO 14|单片机向 MP3 发送播放/音量等十六进制指令。|
||TX (发送)|GPIO 27|(非必需) 读取 MP3 状态。若引脚紧张可不接。|
||VCC|5V|建议接 5V，否则推不动喇叭且底噪大。|
||GND|GND|共地|
||SPK1 / SPK2|外接喇叭|直接连接微型扬声器的正负极。|
||BUSY|GPIO 32|回报音频是否完成播放，作为播放下一条的依据。|

### 交互组件
|模块|模块引脚|ESP32 引脚|备注说明|
| ------- | ------- | ------- | ------- |
|EC11 旋转编码器|VCC|3.3V|接 3.3V，确保信号电平安全|
||CLK (S2)|GPIO 25|旋钮中断引脚 2|
||DT (S1)|GPIO 26|旋钮中断引脚 1|
||SW (KEY)|GPIO 33|旋钮按下确认键|
||GND|GND|共地|

| | GND | 面包板 蓝色接地轨 (GND) | |
